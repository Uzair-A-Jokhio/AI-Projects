# AI Projects

This repository contains a collection of various AI projects, including:

- **`aiagent`**: A toy project demonstrating a simple AI agent.
- **`diabetes-readmission`**: A project focused on predicting diabetes readmission rates, likely using machine learning models.
- **`RAG`**: An implementation of a Retrieval-Augmented Generation (RAG) system.
- **`Sindhistory-RAG`**: A RAG system focused on the history of Sindh.

Each project folder contains a more detailed `README.md` file with specific information about the project.
